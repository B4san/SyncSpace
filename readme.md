# SyncSpace

App web del MVP de SyncSpace, una plataforma de watch‑parties con videollamadas.

## Cómo usar

1. Abre `index.html` en tu navegador favorito (no requiere build ni servidor).
2. Crea o únete a una sala demo, prueba los controles de video, chat y personalización.
3. Consulta la documentación completa en [docs/technical-spec.md](docs/technical-spec.md).
